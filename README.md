---

## 專案介紹

FirePhoenixRadar 是一個實驗性的氣象預測系統。

目標：

- 預測鳳林火燒雲出現機率
- 預測火鳳現象發生時段
- 建立火鳳觀測資料庫
- 發展地方特色氣象觀光

---

## 目前考慮因素 (Factors)

- High cloud cover（高空雲）
- Mid cloud cover（中層雲）
- Low cloud blockage（低層雲遮蔽）
- Visibility（能見度）
- Wind speed（風速）
- Wind gusts（陣風）
- Rain probability（降雨機率）
- Sunset timing（日落時間）
- Typhoon outer circulation（颱風外圍環流）
